## 1. Introduction and Context

## 2. Methods Summary

### 2.1 IO Table Construction

Supply-Use Tables were converted to symmetric product × product IO tables using the **Product Technology Assumption (PTA)**:

```
D = V / q         [market share matrix,   products × industries]
Z = U · Dᵀ        [intermediate flows,    products × products]
A = Z / x         [technical coefficients, bounded 0 < col_sum(A) < 1]
L = (I − A)⁻¹    [Leontief inverse,       Hawkins-Simon ρ(A) < 1 confirmed]
```

The Hawkins-Simon condition ρ(A) < 1 is verified for all three study years (see Section 3.1). Several SUT products required `clean_a_matrix()` correction where raw data produced column sums ≥ 1.0 (notably crude petroleum in FY 2021–22 where intermediate inputs exceeded total output by 55%, likely a data recording error). Corrected columns were rescaled to A_sum = 0.95, a standard IO repair method that preserves relative input shares while guaranteeing invertibility.

### 2.2 EEIO Water Coefficient Assignment

EXIOBASE 3.8 blue water coefficients (m³/EUR million) were converted to m³/₹ crore using annual EUR/INR exchange rates. A 163-sector EXIOBASE → 140-sector SUT concordance was applied. Sectors with zero EXIOBASE coefficients (primarily services) receive W = 0; agricultural water enters through the Leontief supply-chain propagation.

### 2.3 Tourism Demand (Y vector)

India's TSA 2015–16 was extrapolated to 2019 and 2022 using NAS Statement 6.1 real GVA growth rates (constant 2011–12 prices), multiplied by the CPI deflator to restore nominal ₹ crore values consistent with SUT data:

```
Y_nominal(year) = Y_base_2015 × real_GVA_growth(year) × CPI(year)/CPI(2015-16)
```

This approach follows Temurshoev & Timmer (2011) and OECD satellite account practice. See Section 4 for year-by-year results.

### 2.4 TWF Computation

#### Indirect TWF (supply-chain embedded water)
```
TWF_indirect = W · L · Y
```
where W is the water coefficient vector (m³/₹ crore, shape 140), L is the Leontief inverse (140×140), and Y is the tourism demand vector (₹ crore, shape 140).

**Worked example (3-sector illustration):**
```
W = [5,000  m³/crore,   0  m³/crore,   0  m³/crore]   (only Agriculture has water)
L = [[1.20,  0.80,  0.10],    ← Agriculture multipliers per unit of each sector's output
     [0.30,  1.50,  0.20],    ← Manufacturing multipliers
     [0.40,  0.60,  1.30]]    ← Services multipliers

Y = [0  crore,   150  crore,   300  crore]   (tourists buy Manufacturing and Services)

WL  = W @ L = [5000×1.20 + 0×0.30 + 0×0.40,   5000×0.80 + 0×1.50 + 0×0.60,   5000×0.10 + ...]
    = [6,000  m³/crore,  4,000  m³/crore,  500  m³/crore]    ← Type I water multipliers

TWF = WL · Y = 6,000×0   +   4,000×150   +   500×300
    = 0   +   600,000   +   150,000   =   750,000 m³

Interpretation: although Y[Agriculture] = 0 (tourists buy no raw crops),
agriculture still contributes 750,000 m³ because L propagates upstream demand
through Manufacturing and Services supply chains.
```

This illustrates why agricultural water dominates indirect TWF (60–80%) even though it has zero direct tourism expenditure share.

#### Direct TWF (operational water)
Estimated from activity data (tourist volumes, stays, transport) × published field-study coefficients:
```
Direct_Hotels      = classified_rooms × occupancy_rate × nights_per_year × L/room/night / 1000
Direct_Restaurants = (domestic_days + inbound_days) × meals/day × L/meal / 1000
Direct_Rail        = rail_pkm × tourist_share × L/pkm / 1000
Direct_Air         = air_passengers × tourist_share × L/passenger / 1000
```

#### Total TWF
```
TWF_total = TWF_indirect + TWF_direct_BASE
```

### 2.5 Structural Decomposition Analysis (SDA)

Change in indirect TWF between year-pairs is decomposed using the **two-polar (Dietzenbacher–Los) method**, which eliminates the residual term by averaging two orderings:

```
ΔTWF = W_effect + L_effect + Y_effect  (residual < 0.01%)

W_effect = ½(ΔW·L₀·Y₀ + ΔW·L₁·Y₁)   ← technology / efficiency
L_effect = ½(W₀·ΔL·Y₀ + W₁·ΔL·Y₁)   ← supply-chain structure
Y_effect = ½(W₀·L₀·ΔY + W₁·L₁·ΔY)   ← tourism demand growth
```

### 2.6 Monte Carlo Uncertainty Analysis

10,000 simulations with log-normal sampling (σ = 0.30 for agricultural W, σ = 0.25 for hotel coefficients) and normal sampling (σ = 8% domestic volumes, 5% inbound volumes). Variance decomposition via Spearman rank correlation identifies dominant uncertainty sources.

## 11. Structural Decomposition Analysis (SDA)

Decomposes the change in indirect TWF between year-pairs into three drivers using the two-polar Dietzenbacher–Los decomposition (Dietzenbacher & Los, 1998):

`ΔTWF = W_effect + L_effect + Y_effect`

By construction the three effects sum exactly to ΔTWF (floating-point residual typically <0.001%). The TWF values in the table below are the SDA module's own internal W·L·Y recomputations — these are the only values consistent with the reported effects and may differ slightly from Table 4 (which comes from a separate computation path).

**Reading the % columns:** Each effect is reported as a percentage of |ΔTWF|. When opposing effects are large and nearly cancel — producing a small net ΔTWF — these percentages can exceed ±100% or even ±1000%. Such values are mathematically correct but economically uninterpretable. Rows marked ⚠ have this instability; rely on the absolute bn m³ values instead.
